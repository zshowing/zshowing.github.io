DESIGN

从大学开始，我就在思考到底做什么能带给自己成就感。

也正是从那时开始，由于各种契机，我开始参与一些微小的设计工作，比如学校BBS的进站页面，比如论坛的签名档之类。
正是这些在今天看来微不足道的尝试，让我意识到：创造世间没有的美好事物，是件能让自己全情投入的工作。


CODE

编程不只是一种工具。

它是改造世界的一种手段。

每个男孩子生来就有种控制和驾驭的天性，因此他们会玩电子游戏，会拼积木，长大后喜欢开车——而编程也是这种游戏在成人世界的延续。
